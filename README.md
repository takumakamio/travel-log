## TraveLog　概要
- マップにピンを置くことで地図上に旅の記録ができます
- フォローした人物の旅の記録を確認できます

### サイトテーマ
- MAPBOXを活用し旅の記録を保存、他人に思い出をシェアできるSNSアプリケーション

### ターゲットユーザー
- 旅行の記録をつけたい人

## 開発環境
- スタック：React, Node.js, Express.js, MongoDB
